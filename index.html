Manual do Jogo


Bem-vindo ao manual do jogo "Motos Extremamente Perigosas"! Aqui você aprenderá tudo sobre como jogar, controlar a moto e enfrentar os desafios. Prepare-se para a ação!


  • A moto se move para frente AUTOMATICAMENTE.
  • Pressione A para mover a moto para esquerda.
  • Pressione D para mover a moto para esquerda.
  • Ou use as setas para a esquerda e direita para virar.
  • Troque os pneus antes que eles se desgastem totalmente.
  • Evite obstáculos e atinja o maximo de pontos que conseguir.
